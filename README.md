# Encoder-Only-BERT-Customer-Feedback-Classification
Perform sentiment classification on customer feedback to determine whether the sentiment is positive, negative, or neutral.
Dataset: https://www.kaggle.com/datasets/vishweshsalodkar/customer-feedback- dataset?select=sentiment-analysis.csv 
Objective: Fine-tune a BERT-based model to classify textual feedback according to customer sentiment.
